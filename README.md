# OPTIMA OS

Introduction
OPTIMA OS is a minimalistic operating system designed as a project to demonstrate the core functionalities of modern operating systems. This project includes basic implementations of process management, memory management, file systems, and I/O handling.

# Features
1. Scheduling Algorithm: First Come First Serve (FCFS).
2. Process Synchronization: Implemented using Mutex Locks.
3. Memory Management: Basic allocation and deallocation using `malloc`.
4. InterProcess Communication (IPC): Achieved through shared memory.
5. I/O Handling: Command-Line Interface (CLI) for user input and corresponding outputs.

# File Structure
- src/: Contains source code files for the project.
- docs/: Includes project documentation and diagrams.
- bin/: Compiled binaries for testing.
- README.md: Detailed documentation about the project.

# How to Run
1. Clone this repository:
   
       git clone https://github.com/your-username/OPTIMA_OS.git
   
2. Navigate to the project directory

        cd OPTIMA_OS

3. Compile the code
   
        gcc -o optima_os main.c

4. Run the executable

       ./optima_os

Note: First download and Install GCC Complier, for your preferred OS
